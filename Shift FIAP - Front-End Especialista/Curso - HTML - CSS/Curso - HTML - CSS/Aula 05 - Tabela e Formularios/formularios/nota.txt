Formularios: são dados;

Estrutura:
<form> Componentes aqui </form>

Dentro da tag form, temos alguns atributos especiais:

ID: Identificador, por padrão, mesmo valor do NAME
Name: Nome, por padrão, mesmo valor do ID
Method: forma como os dados serão enviados
Get: Método padrão: Dadaos enviados na barra de URL (Usa em pesquisas, links dinâmicos)
Post: Dados são enviados no corpo dos campos (encapsulados) (Usado em form de contato, cadastro, carrinho de compra: Ddaos mais sensiveis)
Action: O que acontecerá com os dados quando o form for enviado (Aqui tem o endereo de alguém que consegue recuperar os dados digitados e fazer o seu uso)


Campos do Formularios:

98% dos campos, são inseridos através da tag input, seguido pelo atributo type + atributos

São tags proprias select e textarea, e button

- Label vincula um texto a um determinado campo pelo ID do campo.

Form são campos inline, ficam um em baixo do outro
